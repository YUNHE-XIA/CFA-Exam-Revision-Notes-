## 笔记风格 | Note Style

每个 reading 或 module 通常按照以下结构整理：

1. Reading 核心定位  
2. Learning Outcome 对照  
3. 核心概念与定义  
4. 重要公式  
5. 考试导向讲解  
6. 概念对比表  
7. 高频陷阱与误区  
8. 做题思维  
9. 官方题目 / 类似题目讲解  
10. 极简考前记忆版  

Each reading or module is usually organised with the following structure:

1. Core positioning of the reading  
2. Learning outcome summary  
3. Key concepts and definitions  
4. Important formulas  
5. Exam-focused explanations  
6. Concept comparison tables  
7. Common traps and misunderstandings  
8. Question-solving logic  
9. Official-style question explanations  
10. Quick pre-exam memory version

## 为什么建立这个 Repository | Why This Repository

CFA 的内容非常系统，但每个 reading 信息量较大，直接阅读时容易抓不住重点。  
这个 repository 的目标是把长篇 reading 转换成更适合复习和考试记忆的结构化笔记。

The CFA curriculum is detailed and conceptually dense.  
This repository aims to make the material more manageable by converting long readings into structured revision notes.

这些笔记会重点回答：

- 这个 reading 的核心是什么？
- 哪些概念最可能被考？
- 哪些公式必须记住？
- 常见陷阱在哪里？
- 做题时应该如何判断？

The notes focus on:

- What is the core idea?
- What is likely to be tested?
- What formulas must be remembered?
- What are the common traps?
- How should a candidate think when solving questions?

## **使用方式 | How to Use**

可以按照以下方式使用本 repository：

先阅读每个 module 的核心定位；
再看概念解释和公式；
重点复习“高频陷阱”和“做题思维”；
考前快速查看“极简考前记忆版”；
用官方题目或 mock exam 检验理解。

Suggested study approach:

Read the core positioning of each module.
Review the key concepts and formulas.
Focus on common exam traps and question-solving logic.
Use the quick memory version before the exam.
Test understanding with official questions or mock exams.
